# 视觉 Agent 提示词工程

各视觉 agent（角色设计师、场景设计师、分镜师）已内置专业 prompt 工程能力。你在 `message` 中传递**创作意图 + 物理级描述**（三语格式），agent 会自动构建高质量 prompt。

## 三语提示词格式（中文+日语+英语，必须遵守）

**当你调用视觉 agent 时，`message` 必须使用中文、日语、英语三语混合格式。** 中文提供详细语境描述，日语提供专业视觉/动漫术语，英语提供标准 diffusion model 关键词。三语结合能显著提升画面质量和精准度。

### 三语格式公式

```
[中文主体描述（详细动作/环境/光影/情绪）] + [日本語スタイル・品質記述] + [English technical keywords, camera params, quality tags]
```

### 格式详解

**第一段——中文（占 40-50%）**：用中文详细描述主体、动作、环境、情绪、构图意图。

**第二段——日语（占 20-30%）**：用日语补充风格描述、品质标签、氛围词。写实风格用日语摄影术语，动漫风格用日语动画术语。

**第三段——英语（占 30-40%）**：用英语写标准 diffusion model 关键词、相机参数、质量标签。

### 内联标注格式

对于关键概念，可在中文主体描述中使用 **括号内联标注** 同时给出日语和英语：
```
四位程序员（プロジェクト会議／project meeting）围绕会议桌讨论
```

### 完整示例

写实/动漫/多角色/视频等各类三语 message 完整示例见 [references/trilingual-examples.md](references/trilingual-examples.md)。首次构建 message 或切换风格时读取。

## 核心公式（字段顺序严格遵守）

**中文主体描述段**必须按以下顺序组织（重要信息前置，模型对前置词权重更高）：

```
① 主体+动作 → ② 角色锚点复述 → ③ 环境/背景 → ④ 景别/机位/焦距 → ⑤ 光影/色温 → ⑥ 构图（前中远三层） → ⑦ 风格/质感
```

进阶版（电影级）：
```
① 工业级摄影参数 → ② 前景虚化遮挡 → ③ 中景主体与动作 → ④ 背景环境 → ⑤ 胶片/色彩质感
```

**与三语格式的关系**：①-⑥ 写在中文段（内联标注日英关键词），⑦ 风格由日语段承担，英语段负责相机参数和 diffusion 质量标签。三段各司其职，不重复。

**卫生规则**：
- 英文关键词用现在时名词短语，不写完整句子
- 英文关键词段控制在 40-60 词，避免稀释权重
- 在视频 message 中提醒分镜师应用 Negative 通用集：`blurry, deformed, bad anatomy, extra limbs, watermark, text, signature`

## 6 个维度指导

| 维度 | 写法 | 说明 |
|------|------|------|
| **光线三角** | "黄昏暖调逆光 + 光比1:4 + 色温偏暖" | 方向×光比×色温，三要素缺一不可 |
| **焦段景别** | "85mm人像虚化f/1.4" 或 "24mm广角环境叙事" | 焦段决定透视和叙事距离，需与景别匹配 |
| **时间维度** | "冻结在出拳的瞬间" 或 "战斗后的废墟" | 进行时→冲击力，过去时→故事感，将来时→张力 |
| **空间层次** | "前景虚化竹叶遮挡 + 中景月光投影 + 远景雾气" | 前中远三层，打破AI默认的扁平构图 |
| **质感介质** | "Kodak Portra 400色调" 或 "TV动画赛璐璐着色" | 胶片锁色彩，渲染引擎和胶片风不要混用 |
| **微动作** | "手指攥紧、指甲陷入掌心" 而非 "很紧张" | 用物理微动作代替情绪结论，避免AI油腻感 |

## 关键原则

1. **物理参数 > 形容词**：不写 `4k, masterpiece, beautiful`，用焦段/光比/色温/材质等物理参数替代
2. **一个动词 + 方式词**：不堆叠多个动作（`runs, jumps, rolls`），保留一个核心动词 + 方式词（`sprinting, body tilted 30° forward`）
3. **一个 Prompt 一种时态**：不混用过去式和进行时
4. **前景遮挡消除AI味**：加入前景虚化物（树叶/柱子/雾气/镜头水珠）建立机位感
5. **色彩独裁**：主色调占 80%+，辅色调仅在边缘/阴影，不要冷暖50:50
6. **状态快照 > 时间流水账**：不用 `then/after`，描述动作发生时的特定瞬间状态

## 禁止混用的组合

| 冲突组合 | 原因 |
|---------|------|
| 远景 + 浅景深 | 远景需深景深交代环境，逻辑冲突 |
| 广角 + 背景压缩 | 物理相反（广角拉开纵深 vs 长焦压缩空间） |
| 胶片感 + 3D渲染引擎 | Kodak Portra 和 Octane Render 风格冲突 |
| 多个并行动作 | AI 肢体抽搐，用锚点动作+从属微动代替 |

## 景别-表情匹配

| 景别 | 表情描写粒度 |
|------|------------|
| 极特写 ECU | 微观生理动作：瞳孔收缩、皮肤潮红、微颤 |
| 特写 CU | 面部肌肉群：眉心蹙起、唇角压紧、下颚绷紧 |
| 中景 FS | 身体姿态+手势：重心偏移、肩部角度、手指位置 |
| 远景 WS | 轮廓+空间关系：身体倾角、步态、人与环境距离 |

## 内容安全表述规则

描述涉及亲密、暴力或裸露画面时，**传给 agent 的 message** 必须使用物理化/艺术化替代，禁止露骨词汇。此规则与核心公式①-⑦字段顺序共同生效：

- **亲密场景**：用压力/速度/振幅/呼吸节奏等物理量替代生物学直白描写。如 ①"两人额头轻触，呼吸频率同步" ⑤"暖色侧光勾勒面部轮廓"
- **暴力场景**：用力矢量/冲击力/位移等物理描述替代血腥细节。如 ①"拳头接触面积约8cm²，冲击方向右偏15°" ⑥"前景尘土飞扬虚化"
- **裸露处理**：用阴影轮廓/浅景深虚化/布料剪影/蒸汽扩散等艺术手法包裹。如 ⑥"逆光剪影勾勒肩颈曲线，前景蒸汽柔化细节" ④"85mm浅景深f/1.4"
