# 视频成片制作流程

## ⛔ 铁律：禁止跳过任何步骤

> **在调用 `call_storyboard_artist` 之前，你必须已经完成了步骤 1-4（规划、编剧、角色、场景）。**
> **在调用视频生成之前，你必须已经完成了步骤 1-6.5（包括分镜静帧 + 九宫格拆分）。**
> **如果你发现自己想直接跳到分镜或视频生成，停下来，回去完成前面的步骤。**
>
> 每一步完成后，发一行简短状态（如"✅ 完成"），然后**立即执行下一步**，不要等用户回复。

## 流程检查清单（每次生成视频前必须自查）

在执行任何视频生成步骤之前，确认以下 5 项全部完成：
- [ ] ✅ 已调用 `call_screenwriter` 生成剧本
- [ ] ✅ 已调用 `call_character_designer` 生成角色设定图
- [ ] ✅ 已调用 `call_scene_designer` 生成场景图
- [ ] ✅ 已调用 `call_storyboard_artist` 生成分镜静帧（仅静帧，非视频）
- [ ] ✅ 已调用 `split_storyboard_grid` 拆分九宫格（如果分镜是九宫格合成图）

**如果前 4 项任一未完成，禁止跳到后续步骤。第 5 项强烈推荐但非强制。**

## 标准 9 步流程

### 1. 规划
```
→ plan_video_segments(60)  // 用户指定的时长
```
告诉用户分段方案（如 4段×15秒）。**每段视频时长只能为 5、10 或 15 秒**，不要使用其他值。

### 2. 编写剧本（必须！）
```
→ call_screenwriter(workspace_id, "写一个{时长}秒的{主题}短片剧本，分{N}幕，每幕约15秒...")
```
返回后立即进入下一步。

### 3. 设计角色（必须！综合角色设定图）
```
→ call_character_designer(workspace_id, "根据剧本设计主要角色，生成综合角色设定图（一张图包含正面/侧面/背面全身+脸部/头部/眼部/脚部特写）...")
```
返回后立即进入下一步。**如果用户上传了参考图，必须在 message 中注明"用户提供了参考图片，请优先参考用户图片保持人物身份一致"**。

### 4. 设计场景（必须！）
```
→ call_scene_designer(workspace_id, "根据剧本设计各场景环境...")
```
返回后立即进入下一步。

### 5. 角色动作图（可选 — 产品设计师）
**如果剧本涉及人物动作场景，让产品设计师基于角色设定图生成动作图——保持人物身份一致。**
```
→ call_action_designer(workspace_id, "基于已有角色设定图，为第1幕生成角色动作场景：{动作描述}。保持人物身份完全一致。")
```
可为多个关键幕并行调用。返回后立即进入分镜。

### 6. 生成分镜静帧（必须！）
**禁止跳过此步骤。** 分镜师会生成九宫格关键帧图片，后续视频生成会以此为画面参考（first_frame），确保视频画面与角色/场景一致。
```
→ call_storyboard_artist(workspace_id, "只生成分镜静帧关键帧，不要生成视频。参考已有角色和场景，为每一幕生成对应的分镜画面：第1幕={描述}，第2幕={描述}...")
```
返回后立即进入拆图步骤。

### 6.5 拆分九宫格（推荐）
分镜师通常返回九宫格合成图（多张关键帧拼在一张图里）。拆分后，每张独立的分镜图可作为对应视频段的**首帧参考**，显著提升画面一致性。
```
→ get_current_session_assets(workspace_id)  // 获取分镜图 URL
→ split_storyboard_grid(分镜九宫格图片URL)
```
拆分结果会自动推送到画板（1-3 分钟），无需等待。提交拆图后**立即进入视频生成**。

**注意**：仅对有明确网格边界的合成图使用此工具。如果分镜师返回的是单独的分镜图（非九宫格合成），跳过此步直接进入视频生成。

### 7. 生成视频（逐段）
每段视频时长**只能为 5s、10s 或 15s**，默认使用 15s。不要使用 4s、6s、8s 等其他值。

**优先使用拆分后的单张分镜图**：如果已执行步骤 6.5 拆图，在 message 中指定对应的单张分镜图作为画面参考（而非整张九宫格），画面一致性更好。

第1段：
```
→ call_storyboard_artist(workspace_id, "生成第1段视频(15秒)：{第1幕具体画面描述}。参考前面生成的分镜静帧（使用拆分后的第1张分镜图作为首帧参考）。")
```
第2段起：
```
→ extract_video_frame(上一段视频URL, "last")  // 取上段末帧
→ call_storyboard_artist(workspace_id, "使用上一段最后一帧作为开头画面，生成第2段视频(15秒)：{第2幕画面描述}。同时参考拆分后的第2张分镜图。")
```

### 8. 配乐
```
→ call_sound_designer(workspace_id, "为这个{主题}视频配乐，{时长}秒，{氛围}风格")
```

### 9. 确认成果
```
→ get_workspace_assets(workspace_id)
```
向用户报告制作完成。

## 视频生成规则

**用户已在消息中明确要求生成视频时，直接执行，不需要额外确认。** 用户说了"生成视频"、"做视频"就代表已经同意了。

在执行视频生成之前，确保前面所有步骤（编剧、角色、场景、分镜）已完成即可。不要停下来问用户"是否确认"——直接生成。

## 视频生成必须有分镜参考图

**在调用 `call_storyboard_artist` 生成视频之前，必须确保已有对应的分镜图。** 流程要求：
1. **先调用** `call_storyboard_artist` 生成**静态分镜图**（message 中写明"只生成分镜静帧、不要生成视频"）
2. 确认分镜图已成功生成（通过返回结果确认）
3. **再调用** `call_storyboard_artist` 生成视频（此时内部会自动使用分镜图作为首帧参考）

**禁止在没有分镜图的情况下直接生成视频。** 没有分镜图参考的视频会退化为纯文本生成（text2video），画面质量和连贯性大幅下降。
